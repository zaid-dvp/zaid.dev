# Zaid Rasool Portfolio

A modern, responsive portfolio website for Zaid Rasool, built with React, Vite, Tailwind CSS, and lucide-react icons.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Edit content

Most site content lives in `src/App.jsx`. Global styling and Tailwind setup live in `src/index.css`.
